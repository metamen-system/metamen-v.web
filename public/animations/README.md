# Lottie Animations

10 archivos JSON de animaciones Lottie para cinemáticas y hitos.

Renderizados con Lottie React ^2.4.0.

## Archivos esperados

1. death.json — Animación de muerte del avatar
2. resurrection.json — Animación de resurrección
3. level_up.json — Animación de subida de nivel
4. contract_seal.json — Sello de contrato (onboarding juramento)
5. penalty_seal.json — Sello de castigo (onboarding penalty)
6. heart_break.json — Pérdida de HP
7. judgement_success.json — Juicio nocturno exitoso (≥80%)
8. judgement_failure.json — Juicio nocturno fallido (<80%)
9. streak_fire.json — Efecto de racha activa
10. protocol_complete.json — Completar el protocolo de 100 días

## Formato: JSON (Lottie)

## Uso: Teatro de la Consecuencia, hitos, transiciones
